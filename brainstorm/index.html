Expand on Interests & Skills

  • What’s a tech-related topic you’re passionate about? (e.g., AI, cybersecurity, web dev, social impact, video games)
  • Is there a problem you’ve noticed in your daily life that tech could help solve?
  • Have you learned something in class or an internship that you want to explore further?

Software & Development

  • Build a web or mobile app to solve a problem in your community
  • Create a browser extension or automation tool for personal productivity
  • Develop a game (2D, text-based, multiplayer, etc.)

Data Science & AI

  • Analyze trends in a dataset related to sports, climate, or social media
  • Train a machine learning model for image recognition, sentiment analysis, or anomaly detection
  • Create a chatbot or AI-powered assistant for a specific task

Cybersecurity & Ethical Hacking

  • Research vulnerabilities in a common application and propose fixes
  • Build a penetration testing tool or encryption algorithm
  • Study phishing techniques and design an awareness campaign

Hardware & IoT

  • Develop a smart home automation project
  • Use Arduino or Raspberry Pi to build a robotics project
  • Create a wearable device for health monitoring

Ethics & Policy

  • Investigate the impact of AI bias in hiring algorithms
  • Analyze privacy concerns in popular apps and propose improvements
  • Research misinformation spread on social media and build a detection tool
