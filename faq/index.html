San Diego Undergraduate Tech Conference

FAQ

Do I have to apply solo or in a team?

We welcome both solo and team projects.

Can I submit a project that is based in another club / lab / organization?

For the vast majority of cases, yes! Please describe your case in your application under the question that asks about whether you are receiving external support.

How can I come up with an idea for a project?

  • What's a topic you're interested in?
  • Is there a problem you've noticed that you could try to solve or automate?
  • Have you learned something in class or an internship that you want to explore further?
  • Is there a creative or artistic spin you could put on a technical product?
  • Is there data you want to analyze or use for a new purpose?

Where can I find datasets?

  • PhysioNet
  • Data.gov
  • Papers with Code
  • UCI Machine Learning Repository
  • FiveThirtyEight Data
  • Awesome Public Datasets
  • Stanford SNAP Data
  • Google Dataset Search
  • Kaggle Datasets
  • Google Trends
  • Reddit Data API

My project scope has changed since my last submission. Can I still participate?

For the vast majority of cases, yes! Big changes can be common parts of unstructured learning and building processes. Just let us know as soon as you can. We are happy to provide support to help you refine the project and work through obstacles.

Can I get an extension / submit late?

We may be able to offer extensions to certain deadlines. Please contact us as soon as possible.
